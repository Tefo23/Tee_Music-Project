# TeeMusic Recording Studio Website

## 📌 Project Overview
This project is a responsive, one-page website designed and developed for **TeeMusic Recording Studio**.  
It was created as part of a student web development assignment, following recommended web design practices.

The website demonstrates clean structure, separation of concerns, responsiveness, and basic interactivity using vanilla JavaScript.

---

## 🌐 Pages / Sections
The website is implemented as a **one-page layout** with the following sections:

- Home
- About Us
- Services
- Artists
- Contact

Navigation links scroll smoothly to each section.

---

## 🛠 Technologies Used
- **HTML5** – semantic structure
- **CSS3** – layout, colours, animations, responsiveness
- **JavaScript (Vanilla)** – interactivity and animations

❌ No frameworks  
❌ No Bootstrap  
❌ No inline or embedded CSS  

---

## 🎨 Design Features
- Fixed header navigation
- Active navigation highlight on scroll
- Responsive container layout
- External stylesheet for all styling
- Scroll-based section animations
- Clean and accessible layout

---

## 🖼 Image Slider Features
- Automatic sliding images
- Pause on hover
- Dots navigation
- Fully responsive
- Implemented using JavaScript and CSS transitions
